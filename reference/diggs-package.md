# diggs: Delineation-Informed Geographic Ground Selection

Interactive Shiny app for selecting historic orthophoto imagery from the
BC Data Catalogue. Draw polygons or upload a custom AOI, filter by
year/media/scale, view estimated photo footprints, and export selections
to CSV. Built on the fly package for spatial photo operations.
